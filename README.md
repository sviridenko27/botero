# Парфюмерный бутик (FastAPI)

Веб-приложение для парфюмерного бутика с публичной витриной и админ-панелью.

## Технологии

- Python 3.10+
- FastAPI
- Jinja2 + HTML/CSS (frontend)
- SQLite
- Docker Compose
- Makefile

## Разделы сайта

- `Ароматы`
- `Бренды`
- `Уход`

Контент для разделов управляется через админ-панель: создание, редактирование, удаление карточек и загрузка изображений файлов.

Сразу под шапкой сайта размещен промо-блок:
- слева логотип `BOTERO`,
- справа слайдер новостей, которые управляются через админ-панель.

## Запуск локально

```bash
make install
make dev
```

Сайт будет доступен на [http://localhost:8000](http://localhost:8000).

## Запуск через Docker

```bash
make docker-build
make docker-up
```

## Админ-панель

- URL: `http://localhost:8000/admin`
- Авторизация: HTTP Basic
- По умолчанию:
  - Логин: `admin`
  - Пароль: `admin123`

Можно переопределить через переменные окружения:

- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`

## Структура

- `app/main.py` — маршруты, админ CRUD, загрузка файлов
- `app/models.py` — модели БД
- `templates/` — HTML-шаблоны
- `static/css/styles.css` — стили с палитрой `#293052` и `#DE9C99`
- `uploads/` — загруженные изображения
- `data/` — SQLite база
